# P&O1b: De precieze plaatser
## Installeren op een Raspberry Pi Pico
1. Reset de Pico door er een nuke-bestand op te zetten.
2. Installeer CircuitPython **9** op de Pico.
3. Download de Pico-files (CircuitPython **9**) van Toledo en plaats de `lib`-map hierin in de root van de Pico.
4. Clone deze repository, verander de naam van de `docs`-map naar `static` en plaats deze in de root van de Pico.
5. Plaats **de inhoud** van de map `besturingsalgoritme` in de root van de Pico.
## Gebruik
1. Verbind met het wifi-netwerk van de Pico.
2. Surf via een webbrowser naar het IP-adres van de Pico op poort `80` (voorbeeld: `192.168.0.1:80`).
